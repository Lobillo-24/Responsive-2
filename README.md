# Responsive-2

Página web estática hecha con HTML5 y CSS3 como práctica del módulo de diseño web responsive.

## ¿Qué es?

Una página sencilla que incluye un menú de navegación, una sección hero, varias secciones de contenido, una galería de imágenes y un footer. También tiene botones para cambiar entre tema claro, oscuro y automático.

## Tecnologías

- HTML5
- CSS3

## Características

- Se adapta a móvil, tablet y escritorio
- Tres temas: claro, oscuro y automático
- Galería de imágenes responsive
- Navegación sticky (se queda fija al hacer scroll)

## Archivos

```
responsive-2/
├── index.html   → estructura de la página
└── styles.css   → todos los estilos
```

## Cómo verlo

1. Descarga o clona el repositorio
2. Abre `index.html` en el navegador

No necesita instalación ni servidor.
